User Guide

Welcome to the DB Backup Manager. This tool is designed to help you manage, automate, and monitor your local database backups with ease.

Dashboard

The Dashboard provides a bird's-eye view of your entire backup system.

  • Total successful vs failed backups.
  • Real-time activity feed.
  • Quick access to recent logs.

Databases

Manage your live connections and perform manual backups.

1. Service Detection

Automatically find running MySQL/MariaDB instances.

2. Manual Backup

Select multiple databases and run an immediate backup task.

Schedules

Set up recurring backup tasks to run automatically.

Choose between Daily, Weekly, or Monthly intervals. You can specify the exact time for the backup to start.

Backups History

Access and manage all previously created backup files.

  • Restore: Quickly restore a database from an older file.
  • Management: Delete old records and physical files.
  • Details: View backup size, date, and status.

Diagnostics

Troubleshoot your environment and connection.

This page runs a series of tests to verify your database connection, file system permissions, and environment setup. Green checks mean you're ready to go!

Settings

Configuration of connection and storage.

Connection: Host, Port, User, Password.
Storage: Select the directory for backups.
Options: Enable/Disable GZIP compression.
